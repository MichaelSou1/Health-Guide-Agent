# 常见健康术语简释（General 私有）

## 用途
- 用户遇到不懂的缩写或专业词时，给出 1-2 句通俗解释。
- 适合寒暄 / 入门阶段的快问快答；不用于专业深度讨论（那应该由对应专家 Agent 处理）。

## 训练类

- **RM（Repetition Maximum）**：某个动作在规范动作下最多能完成 N 次的重量。5RM = 最多能做 5 次的重量。
- **1RM**：一次最大重量。力量训练中常用来设定相对强度。
- **RPE（Rate of Perceived Exertion）**：主观费力度 1-10。RPE 8 = 剩余约 2 次到力竭。
- **Set / Rep**：组 / 次。3×10 = 做 3 组，每组 10 次。
- **超负荷（Progressive Overload）**：通过逐步增加重量、次数或组数让身体持续适应。
- **肌肥大（Hypertrophy）**：肌肉体积增加。
- **Deload（减载周）**：训练量有意减半的恢复周。
- **HIIT**：高强度间歇训练，短时高强度与低强度交替。
- **LISS**：低强度稳态有氧，如轻松慢跑、散步、骑行。
- **Z2 训练**：有氧心率区间 2，能说话但微喘，耐力基础核心区间。

## 营养类

- **TDEE（Total Daily Energy Expenditure）**：每日总能量消耗，等于基础代谢 + 活动消耗 + 食物热效应。
- **BMR（Basal Metabolic Rate）**：基础代谢率，静息状态下维持生命的最低能量。
- **宏量营养素 / 宏（Macros）**：蛋白质、碳水、脂肪三大类。
- **微量营养素（Micros）**：维生素、矿物质。
- **热量赤字 / 盈余**：摄入 < 消耗 为赤字（减脂）；摄入 > 消耗 为盈余（增肌）。
- **GI / GL**：升糖指数 / 升糖负荷，食物对血糖影响大小。
- **IF（Intermittent Fasting）**：间歇性断食，如 16:8。
- **MPS（Muscle Protein Synthesis）**：肌肉蛋白合成。
- **亮氨酸阈值**：触发最大 MPS 所需单餐亮氨酸量（约 2.5-3.0 g）。

## 睡眠 / 恢复类

- **REM**：快速眼动睡眠阶段，与记忆整合、情绪调节相关。
- **慢波睡眠（深睡，N3）**：身体修复、激素分泌主阶段。
- **睡眠效率**：实际睡眠时间 / 卧床时间，理想 > 85%。
- **HRV（Heart Rate Variability）**：心率变异性，副交感活性的代理指标，用于评估恢复。
- **CBT-I**：失眠认知行为治疗，慢性失眠一线推荐。

## 心理类

- **正念（Mindfulness）**：有意地、不评判地对当下保持觉察。
- **认知重构（Cognitive Reappraisal）**：识别并调整思维陷阱。
- **MBSR / MBCT**：正念减压课程 / 正念认知疗法。
- **Burnout（职业倦怠）**：长期工作压力导致的情绪耗竭 + 犬儒 + 成就感下降。

## 体检类

- **BMI**：体重 / 身高²。
- **LDL-C / HDL-C**：低密度 / 高密度胆固醇，俗称"坏/好"胆固醇。
- **HbA1c**：糖化血红蛋白，反映近 2-3 个月血糖平均水平。
- **血压 130/85 mmHg**：前者收缩压（高压），后者舒张压（低压）。
- **DEXA**：双能 X 射线吸收法，测骨密度与身体成分的金标准。

## 本表的使用方式
- 用户问"1RM 是什么" → 用一句话解释即可，不要顺便推销训练计划。
- 用户连问多个术语 → 先回答当前的，再询问 "你是在哪里看到这个词的？"，判断是否需要交接给对应专家。
- 本表会持续扩充，若遇到未收录的术语，优先给出最简短的通俗解释而不是空泛"请联系专家"。
